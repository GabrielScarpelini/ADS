
A instalação do SQLAlchemy é muito simples, basta executar 
no terminal integrado do VSCode (ou no CMD do windows):
pip install --user sqlalchemy 


O sqlite é um sistema de bancos de dados relacionais muito simplificado. 
Guarda todas as informações do banco de dados em um único arquivo, e não precisa 
de nenhuma configuração adicional.
www.sqliteonline.com/
